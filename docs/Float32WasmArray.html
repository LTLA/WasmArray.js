Returns:

A WasmArray containing a view on the specified subarray.

The returned object does not own the memory on the Wasm heap, so free() will not have any effect. It does, however, hold a reference to its parent object, i.e., the WasmArray instance on which view() was called. This reference avoids premature garbage collection of the parent and inadvertent invalidation of the views. Of course, all views will be invalidated if the parent's free() method is invoked manually.
