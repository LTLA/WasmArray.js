convertToWasmArray(space, x, arrayClassopt)

Source:
  • convert.js, line 24

Convert a (Typed)Array into a new WasmArray of the specified subclass.

Parameters:
Name Type Attributes Description
space number

Identifier for the Wasm memory space, produced by register.
x Array | TypedArray

Array to be converted.
arrayClass class <optional>

Desired subclass of the WasmArray.
Returns:

Instance of a WasmArray subclass containing the same contents as x.

If arrayClass is not provided, it defaults to an appropriate type based on x. For TypedArray inputs, this is the WasmArray corresponding to the TypedArray subclass, while for Array inputs, this is the Float64WasmArray subclass.
